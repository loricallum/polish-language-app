# Duolingo-clone

This is a simple clone of the Duolingo app built in React Native, Expo, and Typescript.

## Setup
### Requirements
- Expo
- Xcode
- Android Studio

### Steps
- Clone repo.
- Run `yarn start`
- Choose your platform
